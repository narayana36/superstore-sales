# 📊 Sales Analytics Dashboard (Power BI)

## 🚀 Project Overview

This project analyzes retail sales data to uncover insights related to revenue, profit, and product performance. The dashboard helps in understanding business trends and supports data-driven decision-making.

---

## 🛠️ Tools Used

* Power BI
* Python (Pandas for data cleaning)
* Excel

---

## 📌 Key Features

* KPI tracking (Sales, Profit, Orders, Profit Margin)
* Monthly sales and profit trend analysis
* Category-wise performance analysis
* Region-wise sales and profit breakdown
* Interactive filters (Year, Region, Category)

---

## 🧠 Key Insights

* West region generates the highest sales and profit
* Technology category has the highest profit margin
* Furniture category shows low profitability
* Sales peak during Q4 (Nov–Dec)
* Some products have high sales but low profit
